# Backtracking-as-a-solution-to-open-financial-problems-in-C.
This project offers through the use of backtracking, a new way to tackle open problems within finance. Its goal is to highlight how computer science, can when associated with finance offer intelligent solutions to difficult problems.

# Motivation.
What motivated me behind this project, is the desire to analyse difficult financial problems and link them with computer science in order to solve intelligent solutions using backtracking.
I believe that finance and computer science go along, as many experts suggest but I wanted to try to cross the disciplines together by myself with those implementations.
The backtracking algorithms use side effects in order to even allow finding the right sequence.

# How to use it ?
Download the repository, both src and include folders hold respectively the .c files and the .h headers.
Using those, you can hope for a relatively good implementation within your projects.
The algorithms should be rather easy to use, and there are functions for even making up groups of data as it may be difficult to manage such problems. 
The implementations, ensure that used rightfully you can treat small instances of the problem at hand with great precision.

# Limitations.
Naturally, as with most backtracking algorithms the complexity is the most visible problem as it can grow exponentially as the problem increases in size.
The current implementations, should only be used for small situations involving the problems that are studied.
It is not subject to a concrete practical use and I believe that I lack the material to improve it in a substantial manner currently as I am a student and thus still learning, nevertheless there is a direct
value to studying such implementations.

I hope you will enjoy the implementations.
